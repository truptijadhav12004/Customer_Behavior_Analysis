# Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from raw data processing to business intelligence reporting. The project involves data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, dashboard development, and presentation of insights.

The goal is to transform raw data into meaningful business insights that support data-driven decision-making.

---

## Dataset

The dataset contains business-related records such as customer, sales, product, or transaction data.

### Dataset Features

* Customer Information
* Product Details
* Sales Transactions
* Revenue Metrics
* Regional Data

## Project Steps

### 1. Data Loading

* Imported dataset into Python using Pandas.
* Performed initial data inspection and validation.

### 2. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected data types.
* Standardized column formats.

### 3. Exploratory Data Analysis (EDA)

* Analyzed data distributions.
* Identified trends and patterns.
* Generated summary statistics.
* Created visualizations for better understanding.

### 4. Database Integration

* Loaded cleaned data into PostgreSQL/MySQL.
* Created database tables.
* Established database connections using Python.

### 5. SQL Analysis

Executed SQL queries to:

* Calculate total sales and revenue.
* Identify top-performing products.
* Analyze customer behavior.
* Generate business performance metrics.

### 6. Power BI Dashboard Development

Created an interactive dashboard containing:

* KPI Cards
* Revenue Analysis
* Sales Trends
* Product Performance
* Customer Insights
* Interactive Filters and Slicers

### 7. Reporting & Presentation

* Prepared a detailed project report.
* Created a professional presentation using Gamma.
* Summarized key findings and recommendations.

## How to Run

### Prerequisites

* Python 3.x
* PostgreSQL or MySQL Server
* Power BI Desktop
* Jupyter Notebook

### Steps

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Install required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
   ```

3. Load the dataset into Python.

4. Run data cleaning and EDA notebooks.

5. Load the cleaned data into PostgreSQL/MySQL.

6. Execute SQL queries for analysis.

7. Open the Power BI (.pbix) file to view the dashboard.

8. Review the project report and presentation for detailed insights.
